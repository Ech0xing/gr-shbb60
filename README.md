# gr-shbb60

**WORK IN PROGRESS**. A GNURadio 3.9 OOT (out-of-tree) module for the Signal Hound BB60C USB spectrum analyzer.

## Building

First, install the Signal Hound SDK. Then build the GNURadio module.

```
$ mkdir build
$ cd build
build/$ cmake ..
build/$ make
build/$ sudo make install
build/$ sudo ldconfig
```
